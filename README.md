# Pomodoro-CLI

This is a pomodoro CLI to be more productive using the Pomodoro method.

## How to run
```shell
cargo run 
# Or
caro run --release
```

## How to build
```shell
cargo build
# Or
cargo build --release
```

## Note:
Make sure to install `pulseaudio-alsa`.
